# Разбор demo.mp4 по фазам

Источник: `demo.mp4` — захват **рекламного креатива** Gold and Goblins (AppQuantum), 720×1280, 28 с, 678 кадров (~24 fps), портрет.

> ВАЖНО: это запись рекламы, поверх игры есть хром: внизу баннер «Gold and Goblins — Кликер игра / Google Play» (~120 px), кнопка «Пропустить» (низ-право), метка «Реклама» (низ-лево), крестик закрытия (верх-право). **Игровая область ≈ верхние 0–1080 px.** При сравнении кропать к игре и маскировать оверлеи (см. «Кроп» ниже).

## Фазы по таймкоду

| Фаза | Время | Кадры | Что в кадре |
|---|---|---|---|
| **A. Заставка-ковёр** | 0.0 – ~0.5 c | f_0001 – ~f_0012 | Бесконечный ковёр золота + розово-маджентовые самоцветы, тёмная скальная рамка, дозер в центре. Это «красивая ложь» (ТЗ §2.1) — НЕ геймплей. |
| **B. Геймплей** | ~0.5 – 28 c | ~f_0012 – f_0678 | Лавандовый грунт, синий дозер гребёт ДИСКРЕТНЫЕ крупные монеты в горку, ворота-множители, апгрейд-пады, счётчик банка. Это основная часть ролика = целевая фаза по ТЗ §2. |

Переход A→B — резкий кат около 0.5 c (cell 2 контактного листа уже геймплей).

## Визуальная характеристика геймплея (фаза B)

- **Грунт:** лавандово-фиолетовый, не серый. Матовый.
- **Фон/окружение:** фиолетовый, скалы по краям тёмно-фиолетовые.
- **Монеты:** дискретные, КРУПНЫЕ, тёплое золото с читаемым ободком; в горке видны отдельные монеты (не «ковёр/мёд»). ✓ совпадает с интентом ТЗ §2.2.
- **Дозер:** синий корпус, тёмные гусеницы, нож спереди; world-aligned камера.
- **Ворота:** два визуальных типа —
  - ×10: фиолетовые стойки + светлая/белая полупрозрачная завеса с «×10», сверху ценник (напр. «150»).
  - ×100: крупный **КРАСНЫЙ** шеврон-баннер «×100».
- **Апгрейд-пады:** фиолетовая рампа + белый полупрозрачный «призрак» награды + надпись «UPGRADE».
- **HUD:** крупный счётчик банка сверху-слева с иконкой монеты (напр. «1436», «2500»). Всплывающие «×N».
- **Камера:** изометрия, вид сверху-сзади, довольно крутой угол, не вращается с дозером.

## Кроп для сравнения (применять к РЕФ и рендеру одинаково)

- Игровая область: `y ∈ [0, 1080]`, полная ширина `x ∈ [0, 720]`.
- Маски-оверлеи (исключить из метрик): кнопка «Пропустить» (≈ x 470–690, y 980–1040), «Реклама» (≈ x 0–120, y 1030–1075), крестик закрытия (≈ x 660–720, y 0–60).
- Низ (баннер Google Play, y>1080) — полностью отбросить.

## Канонические keyframes (мишени, фаза B)

- `kf_establish` ← f_0060 — установочный: фиолетовые ворота ×10, небольшая кучка, чистое чтение грунта+ворот.
- `kf_hill` ← f_0240 — красные ворота ×100 + крупная горка монет на ноже (ядро ощущения тяжести).
- `kf_spread` ← f_0420 — широкий разброс монет + апгрейд-пад «UPGRADE».
- `kf_pad` ← f_0600 — ссып у апгрейд-пада, дозер рядом.
